# Load Flow Analysis using ETAP

## Objective
To analyze voltage profile, power flow, and improve system performance using reactive power compensation.

## System Configuration
- Utility Source
- 11 kV Bus
- 2 MVA Transformer (11/0.433 kV)
- 0.433 kV Bus
- 1.6 MVA Load

## Initial Results
- Bus2 Voltage: 0.965 pu
- High reactive power demand observed

## Improvement (Capacitor Bank)
- Added 400 kVAR capacitor at LV bus

## Final Results
- Bus2 Voltage improved to 0.978 pu
- Reactive power demand reduced
- System efficiency improved

## Tools Used
- ETAP 2020
